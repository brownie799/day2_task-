# 🐍 Python Basics: Variables, Data Types, and Functions

A beginner-friendly Python project that introduces the fundamental concepts of Python programming, including **variables**, **data types**, and **functions**. This repository provides simple explanations and practical examples to help learners build a strong foundation in Python.

## 📌 Features
- Easy-to-understand examples
- Covers Python fundamentals
- Well-structured and beginner-friendly code
- Ideal for students and beginners

## 📚 Topics Covered
- Variables
- Data Types
  - Integer
  - Float
  - String
  - Boolean
  - List
  - Tuple
  - Dictionary
  - Set
- Functions

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/python-basics.git
   ```

2. Navigate to the project folder:
   ```bash
   cd python-basics
   ```

3. Run the Python file:
   ```bash
   python filename.py
   ```

## 🎯 Learning Outcomes
- Understand Python syntax
- Use variables and data types effectively
- Write and reuse functions
- Build a strong foundation for advanced Python programming
